## Getting Started

### First Time Setup

4. Upload style reference documents if desired

## Button Reference

### AI Generation Buttons

#### Continue Writing
- **Location**: Main AI toolbar
- **Function**: AI writes the next 2-3 paragraphs from your cursor position
- **When to Use**: When you want the AI to continue your story naturally
- **Tips**: 
  - Place cursor where you want continuation
  - Works best with at least a paragraph of context
  - Review preview before inserting
- **Style Integration**: ✅ Uses style profile automatically

#### Generate Scene
- **Location**: Main AI toolbar
- **Function**: Generates a complete 3-5 paragraph scene
- **When to Use**: When you need a full scene based on plot beats
- **Tips**:
  - Works best with defined plot beats
  - AI selects next uncompleted plot beat automatically
  - Review preview to ensure it matches your vision
- **Style Integration**: ✅ Uses style profile automatically

#### Add Dialogue
- **Location**: Main AI toolbar
- **Function**: Generates 4-8 lines of character dialogue
- **When to Use**: When you need character conversations
- **Tips**:
  - Uses character voice profiles automatically
  - Selects characters from your story
  - Review to ensure voices sound authentic
- **Style Integration**: ✅ Uses style profile and character voices

#### Add Description
- **Location**: Main AI toolbar
- **Function**: Generates 1-2 paragraphs of rich description
- **When to Use**: When you need atmospheric or sensory details
- **Tips**:
  - Focuses on sensory details (sight, sound, smell, touch)
  - Great for breaking up dialogue or action
  - Review to ensure it doesn't slow pacing
- **Style Integration**: ✅ Uses style profile automatically

#### Add Character
- **Location**: Main AI toolbar
- **Function**: Introduces a new character with engaging entrance
- **When to Use**: When a new character enters the scene
- **Tips**:
  - Creates memorable introductions
  - Use Entity Extraction Wizard after to create character profile
  - Can enhance character with Character Enhancement feature
- **Style Integration**: ✅ Uses style profile automatically

#### Review
- **Location**: Main AI toolbar
- **Function**: Analyzes chapter for consistency, pacing, and style issues
- **When to Use**: After writing substantial content
- **Tips**:
  - Reviews last 3000 characters
  - Provides specific, actionable suggestions
  - Review each suggestion individually
- **Style Integration**: ✅ Uses style profile automatically

### Text Selection Tools

These tools appear when you select text in the editor.

#### Rewrite
- **Function**: Rewrites selected text with improved clarity and style
- **When to Use**: When text needs improvement but meaning should stay the same
- **Tips**: Works best with 1-3 paragraphs
- **Style Integration**: ✅ Uses style profile automatically

#### Expand
- **Function**: Adds more detail and description to selected text
- **When to Use**: When text feels too sparse
- **Tips**: Review to ensure it doesn't slow pacing too much
- **Style Integration**: ✅ Uses style profile automatically

#### Make Funnier
- **Function**: Adds wit, absurd details, and comedic timing
- **When to Use**: When text needs humor
- **Tips**: Works best with dialogue and character interactions
- **Style Integration**: ✅ Uses comedy style from profile

#### Make Darker
- **Function**: Adds dread, unease, and horror undertones
- **When to Use**: When text needs more tension or darkness
- **Tips**: Great for building tension
- **Style Integration**: ✅ Uses horror style from profile

#### Interject Entity
- **Function**: Interjects an entity (actor, item, skill, etc.) into selected text
- **When to Use**: When you want to naturally add an entity to a paragraph
- **Keyboard Shortcut**: Ctrl+I (Cmd+I on Mac)
- **Tips**:
  - Select paragraph first
  - Choose entity type and specific entity
  - Select mood for how entity is introduced
  - Review placement options
- **Style Integration**: ✅ Uses style profile automatically

### Mood Rewrite Buttons

Quick one-click mood adjustments for selected text:
- **Comedy**: Funnier, absurd, casual
- **Horror**: Dark, tense, unsettling
- **Tense**: High tension, urgent, fast-paced
- **Relaxed**: Calm, slow, detailed
- **Fast**: Quick pacing, snappy, minimal detail
- **Slow**: Measured, contemplative, rich detail
- **Rich**: Detailed, sensory, immersive
- **Sparse**: Minimal, focused, essential only
- **Intense**: Emotionally charged, high impact
- **Detached**: Clinical, unemotional, formal
- **Dark**: Bleak, ominous, heavy
- **Light**: Bright, optimistic, playful
- **Absurd**: Surreal, ridiculous, comedic
- **Grounded**: Realistic, believable
- **Formal**: Proper, structured, dignified
- **Casual**: Conversational, relaxed

All mood rewrites use style integration automatically.

### UI Toggle Buttons

#### Save & Extract
- **Function**: Saves chapter and extracts entities
- **Keyboard Shortcut**: Ctrl+S (Cmd+S on Mac)
- **When to Use**: Regularly to save progress and track entities
- **Tips**: Opens Entity Extraction Wizard automatically

#### Undo/Redo
- **Function**: Undo/redo last changes
- **Keyboard Shortcuts**: Ctrl+Z / Ctrl+Y (Cmd+Z / Cmd+Shift+Z on Mac)
- **Tips**: Works with both manual edits and AI insertions

#### Focus Mode
- **Function**: Dims everything except current paragraph
- **Keyboard Shortcut**: F
- **When to Use**: For detailed editing of specific sections

#### Fullscreen
- **Function**: Expands to fullscreen writing mode
- **Keyboard Shortcut**: F11
- **When to Use**: For distraction-free writing

## Workflow Guides

### Standard Writing Workflow
1. Open or create a chapter
2. Write initial content (1-2 paragraphs minimum)
3. Use "Continue Writing" to let AI take over
4. Review generated content in preview panel
5. Accept, edit, or regenerate as needed
6. Select text and use tools to refine
7. Save & Extract regularly to track entities
8. Use Review button to check for issues
9. Finalize and save

### Entity Management Workflow
1. Write chapter content mentioning entities
2. Click "Save & Extract"
3. Entity Extraction Wizard opens automatically
4. Review extracted entities:
   - New entities: Create or skip
   - Existing entities: Confirm match or update
   - Upgrades: Review and apply
5. For new characters: Use "Enhance Character" for full profile
6. Complete wizard to save entities

### Style-Consistent Writing Workflow
1. Ensure style profile is set up in Onboarding Wizard
2. Upload style reference documents if desired
3. All AI generation automatically uses style profile
4. Review generated content to ensure style match
5. Regenerate if style doesn't match
6. Use mood rewrites to adjust tone while maintaining style

### Editing Workflow
1. Select text you want to improve
2. Choose appropriate tool (Rewrite, Expand, Mood Rewrite)
3. Review preview panel
4. Accept, edit, or regenerate
5. In Review Mode: Changes are tracked
6. In Draft Mode: Changes are direct

## Feature Tutorials

### Character Enhancement
1. Extract entities from chapter
2. Find new character in Entity Extraction Wizard
3. Click "Enhance Character" button
4. AI generates full profile:
   - Stats based on class/role
   - 3-5 starting skills
   - Story-relevant equipment
   - Full biography
   - 3-5 nicknames
5. Review enhanced profile
6. Click "Apply Enhanced" to create character
7. Character is created with all fields filled

### Entity Interjection
1. Select paragraph where entity should appear
2. Right-click or press Ctrl+I
3. Choose entity type (actor, item, skill, location, event)
4. Select or create entity
5. Choose mood (preset or custom)
6. Click "Generate Interjection"
7. Review placement options:
   - Replace selected text
   - Insert before
   - Insert after
   - Blend into existing
8. Select preferred option
9. Click "Apply Interjection"

### Continuity Checking
1. Select text to check (or leave unselected for recent content)
2. Open Writing Enhancement Features panel
3. Click "Check Continuity"
4. Review flagged issues:
   - Character appearance inconsistencies
   - Item ownership conflicts
   - Stat changes
   - Location issues
   - Timeline problems
5. Apply suggested fixes as needed

### Pacing Analysis
1. Open Writing Enhancement Features panel
2. Click "Analyze Pacing"
3. Review metrics:
   - Average sentence length
   - Average paragraph length
   - Dialogue/action/description ratios
4. Review identified issues
5. Apply suggestions for improvement

### Emotional Beat Tracking
1. Open Writing Enhancement Features panel
2. Click "Track Emotional Beats"
3. Review emotional timeline
4. Identify where to heighten or reduce tension
5. Adjust content accordingly

## Style Integration

### How It Works
All AI generation functions automatically use:
- Style profile from Onboarding Wizard
- Style reference documents (if uploaded)
- Character voice profiles
- Story foundation data
- Negative examples (what to avoid)

### Style Priority
1. **Style Profile**: Primary style guide (tone, voice, humor)
2. **Style References**: Examples of desired writing
3. **User Instructions**: Specific requests in prompts
4. **Character Voices**: Individual character speech patterns

### Ensuring Style Consistency
- Complete Onboarding Wizard thoroughly
- Upload style reference documents
- Review generated content for style match
- Regenerate if style doesn't match
- Use mood rewrites to adjust tone while maintaining style

## Troubleshooting

### AI Not Matching Style
- **Solution**: Check that style profile is complete
- **Solution**: Upload style reference documents
- **Solution**: Regenerate with more specific instructions
- **Solution**: Review negative examples

### Entities Not Extracting
- **Solution**: Ensure chapter has substantial content
- **Solution**: Check that entities are clearly mentioned
- **Solution**: Manually create entities if needed

### Preview Panel Not Showing
- **Solution**: Check that generation completed
- **Solution**: Try regenerating
- **Solution**: Check browser console for errors

### Style Integration Not Working
- **Solution**: Verify `smartContextEngine.buildAIContext()` is being called
- **Solution**: Check that style profile exists
- **Solution**: Ensure style references are uploaded

### Performance Issues
- **Solution**: Limit chapter length for analysis
- **Solution**: Close unused panels
- **Solution**: Clear browser cache
